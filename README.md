# localdoby-installer

## Installation

```
git clone git@github.com:jon004/localdoby-installer.git
cd localdoby-instaler
chmod +x install.sh
./install.sh
```

# Usage Sample

```
// 1. Start LLM Server
~/.localdoby/bin/llmserver

// 2. Run document tools commands
~/.localdoby/bin/document-tools upsert ~/path/to/doc.pdf
```
